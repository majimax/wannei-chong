# 🦋 我是哪种濒危货？ - 项目规格

## 1. Concept & Vision

一个嵌有"濒危物种×性格测试"双重趣味的互动测试网页。用户通过15道接地气的情境题，得到一个"濒危货"结果——既有心理测试的"专属感"，又科普了珍稀动物知识。

视觉风格：自然与俏皮的碰撞。森林绿/濒危红作为主色调，配合手绘感的emoji元素，让"濒危"这个沉重话题变得轻松有趣。

## 2. Design Language

### 色彩
- **主色 Primary**: `#2D5A3D` (森林深绿)
- **次色 Secondary**: `#4A7C59` (苔藓绿)
- **强调色 Accent**: `#E85D4C` (濒危警示红)
- **背景色 Background**: `#F5F0E8` (羊皮纸暖白)
- **文字色 Text**: `#2C2C2C` (墨黑)
- **卡片色 Card**: `#FFFFFF`
- **渐变**：从 `#2D5A3D` 到 `#1A3A25`

### 字体
- **标题**: "ZCOOL KuaiLe" (站酷快乐体), cursive
- **正文**: "Noto Sans SC", sans-serif
- **emoji/动物**: 系统原生

### 动效
- 题目切换: 滑入 + 淡入，`400ms ease-out`
- 选项悬停: 轻微放大 `scale(1.02)`，阴影加深
- 结果揭晓: 大emoji弹入 + 卡片依次淡入 (stagger 150ms)
- 进度条: 平滑过渡 `300ms`

## 3. Layout & Structure

### 页面流
1. **封面页** - 标题 + 一句话简介 + 开始按钮
2. **题目页** (×15) - 进度条 + 题目 + 4个选项
3. **结果页** - 动物大图 + MBTI类型 + 名称 + 濒危数据 + 搞笑描述 + 分享按钮

### 响应式
- 移动优先，最大宽度 `480px` 居中
- 选项在小屏垂直堆叠

## 4. Features & Interactions

### 答题
- 点击选项立即选中，自动跳转下一题
- 进度条实时更新
- 不可回退（避免纠结）

### 结果计算
- 每题A/B/C/D对应不同字母组合:
  - E/I: A/B→E, C/D→I
  - S/N: A/B→S, C/D→N
  - T/F: A/B→T, C→T, D→F
  - J/P: A→J, B→P, C→P, D→J
- 统计每个维度的字母计数，取多的
- 组合成16种MBTI结果之一

### 结果展示
- 动物 emoji 大图 (使用对应动物emoji)
- MBTI类型标签
- 动物俗名 + 学名/种群数据
- "习性小知识"卡片
- "为什么是你"搞笑描述
- 经典台词 (引用样式)
- IUCN濒危等级 + 全球数量

### 分享
- 复制结果文案到剪贴板
- 可直接分享到微信/朋友圈

## 5. Component Inventory

### 封面卡片
- 大标题 + emoji
- 副标题说明
- 开始按钮 (主色圆形按钮)

### 进度条
- 顶部固定
- 显示 "Q3/15" + 进度条

### 选项卡片
- 默认: 白底 + 细边框
- Hover: 阴影加深 + 轻微放大
- 选中: 主色边框 + 背景浅绿

### 结果卡片
- 动物emoji (font-size: 80px)
- 类型标签 (圆角pill)
- 名称 + 描述区
- 引用块 (经典台词)

## 6. Technical Approach

- **纯前端**: HTML + CSS + Vanilla JS
- **无框架依赖**: 单文件部署到 GitHub Pages
- **数据**: JS对象内嵌16种结果
- **状态管理**: 简单全局变量 (currentQuestion, answers[])
- **URL**: 刷新重置，无深层链接需求
